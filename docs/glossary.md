# Glossary

**Hover State**  
The design of an element on a page when the cursor is hovering above it.

**Semantic Tags**  
HTML tags that describe the purpose of the code to both the browser and the developer.

**SEO**  
Search Engine Optimization; The practice of improving the visibility and performance of websites and web pages in search engine results pages. ("Search engine optimization", Wikipedia)

**Wireframe**  
A basic sketch depicting how a website will be laid out consisting only of shapes to define the space.

**Hi-Fi / Lo-Fi**  
Abbreviation of high-fidelity / low-fidelity, usually in regards to a mock-up or prototype.

A low-fidelity mock up is one in which only the general placement of things are laid out, with no images or text.

A high-fidelity mockup is one that is visually closest to the final product, with the proper images and text in place.
